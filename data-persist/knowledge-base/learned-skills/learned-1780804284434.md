# 策划案状态确认

在策划案完成后进行状态确认，生成状态确认报告。包含：策划案基本信息（名称/状态/评分）、10章节内容概要表格（章节/内容/状态）、数值验证结果、当前状态汇总、下游团队通知。用于策划阶段完成后确认状态并过渡到开发阶段。
