# 策划案审核

对游戏策划案进行专业审核，生成标准审核报告。包含6维度评分体系（文档规范10分、内容完整20分、设计清晰20分、可执行性20分、数值合理15分、创新性15分），三级审核意见（必须修改、建议修改、优点），风险评估，以及上游数值验证结果集成。70分以上通过，70分以下打回。
