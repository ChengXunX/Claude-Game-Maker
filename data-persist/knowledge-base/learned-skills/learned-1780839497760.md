# level-content-design

为游戏设计具体的关卡内容数据，包含5大核心模块：1)难度曲线建模（多维度难度参数定义+递增策略+ASCII曲线图）2)关卡配置表（每关的具体参数：类型数/层数/行列数/总方块数/目标分，输出可直接使用的JavaScript配置数组）3)解锁机制（前置条件配置+isLevelUnlocked检查函数）4)奖励系统（星级评价阈值表+calculateStars评分函数）5)进度存储（UserProgress数据结构+localStorage持久化save/load函数+默认进度初始化）。每模块输出可直接复制到项目的代码，附带验收标准和测试用例。
