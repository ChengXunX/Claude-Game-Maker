# game-backend-design

为游戏项目设计完整后端系统，包含5大核心模块：用户认证系统（注册/登录/Token管理）、游戏数据持久化（进度存储/多设备同步）、排行榜系统（全球/关卡/好友排行）、成就系统（条件触发/解锁逻辑）、道具系统（库存/使用/购买）。针对游戏场景优化高并发和实时性需求。
