# 射击游戏模板

纯 HTML5 + JavaScript 实现的太空射击游戏，无任何第三方依赖。

## 游戏特性

- **玩家控制**：方向键/WASD 移动飞船
- **自动射击**：飞船自动发射子弹
- **敌人生成**：敌人从屏幕顶部不断出现
- **碰撞检测**：子弹击中敌人、敌人碰到玩家
- **分数系统**：消灭敌人获得分数
- **视觉效果**：星空背景、发光子弹、旋转敌人

## 操作方式

| 按键 | 功能 |
|------|------|
| ← → 或 A D | 左右移动 |
| ↑ ↓ 或 W S | 上下移动 |
| R | 重新开始（游戏结束时） |

## 快速开始

直接在浏览器中打开 `index.html` 即可开始游戏。

## 技术实现

- **渲染**：Canvas 2D API
- **游戏循环**：requestAnimationFrame
- **碰撞检测**：圆形碰撞检测
- **视觉效果**：径向渐变发光、旋转动画

## 自定义配置

修改 `js/config.js` 中的常量可以调整游戏参数：

```javascript
const PLAYER_SPEED = 5;     // 玩家移动速度
const BULLET_SPEED = 8;     // 子弹速度
const ENEMY_SPEED = 2;      // 敌人移动速度
const FIRE_RATE = 200;      // 射击间隔（毫秒）
const ENEMY_SPAWN_RATE = 1500; // 敌人生成间隔（毫秒）
```

## 游戏机制

- 玩家飞船自动向上发射子弹
- 敌人从屏幕顶部生成并向下移动
- 敌人会左右摆动增加难度
- 消灭一个敌人获得10分
- 被敌人碰到则游戏结束
- 敌人离开屏幕底部会自动消失
