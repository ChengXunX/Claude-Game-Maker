# 三消游戏模板

## 游戏介绍

这是一个基于 HTML5 Canvas 的经典三消游戏模板。玩家通过交换相邻的方块，使得三个或更多相同颜色的方块连成一线即可消除得分。游戏支持连锁消除（Combo），并有流畅的动画效果。

## 操作说明

1. **选择方块**：点击任意方块将其选中，选中的方块会有金色高亮边框
2. **交换方块**：选中一个方块后，点击相邻的方块（上下左右）进行交换
3. **取消选择**：点击已选中的方块或点击棋盘外区域取消选择
4. **重新开始**：点击右上角的"重新开始"按钮重置游戏

## 游戏规则

- 交换两个相邻方块，使得横或竖方向出现 3 个或更多相同颜色的方块
- 匹配的方块会被消除，上方的方块会下落填补空位
- 空位会被新生成的随机方块填充
- 如果新方块形成了新的匹配，会自动触发连锁消除（Combo）
- 每次消除都会获得分数，连锁消除会获得额外的分数加成
- 如果没有可用的移动，棋盘会自动重新洗牌

## 技术特点

### 纯 JavaScript 实现
- 不依赖任何第三方库或框架
- 使用原生 ES6+ 语法和类（Class）

### Canvas 2D 渲染
- 使用 HTML5 Canvas 2D API 进行图形渲染
- 支持圆角矩形、渐变、阴影等视觉效果
- 流畅的动画效果（requestAnimationFrame）

### 响应式设计
- CSS 媒体查询适配不同屏幕尺寸
- 触摸事件支持移动端设备

### 游戏功能
- 8x8 网格棋盘
- 6 种不同颜色的方块
- 匹配检测算法（水平和垂直方向）
- 方块下落和填充机制
- 连锁消除（Combo）系统
- 分数计算和显示
- 无可用移动时自动洗牌

## 配置说明

游戏配置集中在 `js/config.js` 文件中，可以调整以下参数：

| 参数 | 说明 | 默认值 |
|------|------|--------|
| GRID_ROWS | 网格行数 | 8 |
| GRID_COLS | 网格列数 | 8 |
| BLOCK_SIZE | 方块尺寸（像素） | 60 |
| BLOCK_GAP | 方块间距（像素） | 4 |
| BLOCK_TYPES | 方块类型数量 | 6 |
| COLORS | 方块颜色数组 | 6 种颜色 |
| SWAP_SPEED | 交换动画时长（毫秒） | 200 |
| DROP_SPEED | 下落动画时长（毫秒） | 150 |
| MATCH_FLASH_DURATION | 消除动画时长（毫秒） | 300 |
| MIN_MATCH | 最小匹配数量 | 3 |
| BASE_SCORE | 基础分数（每个方块） | 10 |
| COMBO_MULTIPLIER | 连击分数倍率 | 1.5 |

## 扩展建议

1. **关卡系统**：添加关卡目标（如达到指定分数、消除特定数量的方块等）
2. **特殊方块**：添加特殊方块（如炸弹、彩虹方块等）
3. **音效**：添加交换、消除、连击等音效
4. **排行榜**：使用 localStorage 保存最高分
5. **主题切换**：支持多种视觉主题
6. **道具系统**：添加可使用的道具（如提示、重排等）
