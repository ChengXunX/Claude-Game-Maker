审查规则

配置此仓库的代码审查规则。不同类型的仓库可以有不同的审查标准。

- 文档格式规范性
- 内容完整性
- 版本更新记录
- 代码规范检查
- 安全性检查
- 性能考量
- 单元测试覆盖
- UI/UX规范
- 性能优化
- 兼容性检查
- 通用代码规范
- 基本质量检查
说明
Agent分配
只有分配到此仓库的Agent才能提交代码审查。 Agent会根据仓库类型使用不同的审查标准。
自动审查
启用后，Agent提交代码会自动触发审查。 审查规则根据仓库类型自动应用。
审查驳回
审查不通过时，Agent会收到驳回通知， 并需要根据审查意见修改代码后重新提交。
