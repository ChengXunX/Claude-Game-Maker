使用说明
多仓库支持

一个项目可以包含多个Git仓库，例如：
- 策划文档仓库（design）
- 服务端代码仓库（server）
- 客户端代码仓库（client）
- 共享库仓库（shared）

Agent分配

每个仓库可以分配多个Agent：
- 策划Agent → 策划文档仓库
- 服务端Agent → 服务端仓库
- 客户端Agent → 客户端仓库
Agent只能提交到分配的仓库

代码审查

每个仓库可以配置独立的审查规则：
- 启用/禁用自动审查
- 针对仓库类型的审查标准
- 审查结果按仓库分类
- 支持仓库级别的审查统计
