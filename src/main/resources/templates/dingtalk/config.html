配置指南
如何获取Webhook URL？
  1. 打开钉钉，进入目标群
  2. 点击群设置 > 智能群助手
  3. 添加机器人 > 自定义
  4. 设置机器人名称和安全设置
  5. 复制Webhook地址
安全设置说明
  • 自定义关键词：消息包含指定关键词才能发送
  • 加签：使用密钥进行签名验证
  • IP地址（段）：限制来源IP
推荐安全设置

建议使用"加签"方式，将密钥填入上方配置框。 或者使用"自定义关键词"，设置关键词为"ChengXun"。

支持的消息类型
  • 文本 简单文本消息
  • Markdown 支持格式化文本
  • 链接 带图片的链接消息
  • ActionCard 带按钮的卡片消息
  • FeedCard 多条链接卡片
通知模板

钉钉通知使用以下模板：

  • ALERT_DINGTALK - 告警通知
  • RECOVERY_DINGTALK - 恢复通知
  • TASK_DINGTALK - 任务通知
  • AGENT_DINGTALK - Agent通知
