Token 用途

API Token 用于 Agent 调用 AI 服务。每个 Token 可以分配给一个或多个 Agent。

状态说明
  • 活跃 - 正常使用中
  • 耗尽 - 配额已用完
  • 禁用 - 手动禁用
  • 过期 - 已过有效期
分配说明

分配 Token 给 Agent 后，该 Agent 将使用此 Token 进行 API 调用。 制作人 Agent 可以在运行时为其他 Agent 分配 Token。
