# 植物设计技能

## 技能描述
设计和平衡游戏中的植物角色，包括属性配置、技能设计、视觉效果规划。

## 触发关键词
植物设计, 植物平衡, 植物技能, 新植物, 植物属性

## 设计流程

### 1. 概念设计
- 确定植物的定位（攻击/防御/生产/控制/支持）
- 设计核心玩法机制
- 规划视觉风格

### 2. 属性配置
```json
{
  "name": "植物名称",
  "sunCost": 100,
  "health": 300,
  "damage": 20,
  "attackSpeed": 1.5,
  "range": "straight",
  "category": "attack"
}
```

### 3. 技能设计
- 主动技能：有冷却时间的主动释放技能
- 被动技能：自动触发的技能效果
- 组合技能：与其他植物配合的技能

### 4. 平衡性检查
- 计算性价比是否合理
- 检查是否过于强大或弱小
- 确保有克制关系

## 设计模板

### 攻击类植物模板
```
基础属性：
- 阳光成本：100-200
- 生命值：300
- 伤害：20-40
- 攻速：1.0-2.0

特殊能力：
- 可选：穿透、范围、减速、灼烧
```

### 防御类植物模板
```
基础属性：
- 阳光成本：50-150
- 生命值：2000-8000
- 伤害：0
- 攻速：0

特殊能力：
- 可选：反弹、治疗、护盾
```

### 生产类植物模板
```
基础属性：
- 阳光成本：50-100
- 生命值：300
- 阳光产出：25-50
- 产出间隔：20-30秒

特殊能力：
- 可选：加速产出、双倍产出
```
