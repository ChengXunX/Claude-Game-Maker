# 植物大战僵尸游戏开发知识库

## 核心设计理念

### 1. 阳光经济系统
- 阳光是游戏的核心资源
- 向日葵是经济基础，需要合理种植数量
- 阳光掉落频率影响游戏节奏
- 设计原则：前期紧张，中期平衡，后期充裕

### 2. 植物克制关系
- 攻击植物克制普通僵尸
- 寒冰射手克制快速僵尸
- 高坚果克制冲锋僵尸
- 樱桃炸弹克制密集僵尸群
- 设计原则：每种植物都有其独特价值

### 3. 僵尸设计原则
- 僵尸应该有明确的威胁等级
- 特殊僵尸需要特定植物克制
- Boss僵尸应该有多阶段战斗
- 波次设计要有节奏感

## 技术实现要点

### 1. 碰撞检测
- 使用网格系统简化碰撞检测
- 植物和僵尸的碰撞范围要精确
- 投射物需要考虑飞行轨迹

### 2. AI寻路
- 僵尸使用简单的直线寻路
- 特殊僵尸需要特殊寻路逻辑
- 考虑植物阻挡和绕行

### 3. 性能优化
- 使用对象池管理子弹和特效
- 合理使用批处理渲染
- 避免频繁的GC

## 平衡性设计

### 植物性价比公式
```
性价比 = (伤害 * 攻速 + 特殊价值) / 阳光成本
```

### 僵尸威胁值公式
```
威胁值 = 生命值 * 速度 * 特殊能力系数
```

### 关卡难度曲线
```
难度 = 基础难度 + (关卡数 * 难度系数) + Boss加成
```
