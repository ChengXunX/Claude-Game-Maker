# 植物大战僵尸游戏模板

## 概述

这是一个类似植物大战僵尸的塔防游戏模板，支持玩家 DIY 自定义植物。模板包含完整的游戏架构设计、植物组件系统、DIY 编辑器设计和分享系统。

## 核心特色

### 1. 植物 DIY 系统
- **组件化设计**: 植物由可组合的组件构成
- **可视化编辑器**: 拖拽式植物设计
- **模板系统**: 基于模板创建和修改植物
- **分享系统**: 玩家可以分享自己的植物设计

### 2. 游戏玩法
- **塔防策略**: 在草坪上放置植物抵御僵尸
- **阳光经济**: 收集阳光来种植植物
- **波次挑战**: 僵尸分波次进攻
- **关卡设计**: 多种场景和难度

## 植物组件系统

### 基础组件

| 组件 | 说明 | 属性 |
|------|------|------|
| AppearanceComponent | 外观组件 | 精灵图、动画、特效 |
| HealthComponent | 生命组件 | HP、护甲、回复 |
| AttackComponent | 攻击组件 | 伤害、攻速、范围 |
| SkillComponent | 技能组件 | 主动技能、被动技能 |
| SunComponent | 阳光组件 | 产出阳光、消耗阳光 |

### 高级组件

| 组件 | 说明 | 属性 |
|------|------|------|
| ProjectileComponent | 投射物组件 | 子弹类型、弹道 |
| AreaComponent | 范围组件 | AOE范围、作用区域 |
| ControlComponent | 控制组件 | 减速、眩晕、击退 |
| BuffComponent | 增益组件 | 攻击加成、防御加成 |
| SpecialComponent | 特殊组件 | 自定义特殊效果 |

## 默认植物模板

### 生产类
- **向日葵**: 产出阳光的基础植物
- **双子向日葵**: 产出更多阳光
- **阳光蘑菇**: 夜间产出阳光

### 攻击类
- **豌豆射手**: 发射豌豆的基础攻击植物
- **双发射手**: 一次发射两颗豌豆
- **三线射手**: 同时攻击三条线路
- **机枪豌豆**: 高速连射

### 防御类
- **坚果墙**: 高血量的防御植物
- **高坚果**: 更高的坚果墙
- **南瓜头**: 保护其他植物

### 控制类
- **寒冰射手**: 减缓僵尸移动
- **冰冻蘑菇**: 冻结全场僵尸
- **忧郁蘑菇**: 周期性眩晕

### 支持类
- **火炬树桩**: 增强经过的豌豆
- **叶子保护伞**: 防御空中攻击
- **花盆**: 在屋顶放置植物

### 特殊类
- **樱桃炸弹**: 范围爆炸
- **土豆雷**: 地雷陷阱
- **大嘴花**: 吞噬僵尸

### 模板系统
- **基础模板**: 预设的植物模板
- **继承修改**: 基于模板创建变体
- **自定义保存**: 保存为新模板
- **模板分享**: 分享到社区

### 验证系统
- **属性检查**: 验证属性值范围
- **平衡性验证**: 检查植物是否过于强大
- **资源检查**: 验证资源引用
- **兼容性测试**: 测试与其他植物的兼容性

## 分享系统

### 功能特性
- **作品上传**: 上传自己的植物设计
- **作品展示**: 浏览其他玩家的作品
- **点赞评论**: 社区互动功能
- **热门排行**: 热门植物排行
- **官方推荐**: 精选植物推荐
- **导入导出**: JSON格式导入导出

## 关卡设计

### 场景类型
- **白天草坪**: 基础场景，阳光充足
- **夜晚草坪**: 阳光减少，需要阳光蘑菇
- **泳池**: 增加水路，需要水生植物
- **浓雾**: 视野受限，需要路灯花
- **屋顶**: 斜面地形，需要花盆

## 技术架构

### 客户端

- **语言**: Java/Python/Go
- **数据库**: MySQL/Redis
- **通信**: WebSocket/HTTP

### DIY系统
- **编辑器**: Electron/Web
- **组件系统**: ECS架构
- **存储**: JSON/云端
- **分享**: REST API
