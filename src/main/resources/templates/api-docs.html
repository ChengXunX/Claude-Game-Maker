API分类

  • GET /agents - 获取所有Agent
  • GET /agents/{id} - 获取Agent详情
  • POST /agents - 创建Agent
  • PUT /agents/{id} - 更新Agent
  • DELETE /agents/{id} - 删除Agent

  • GET /performance/api/all - 获取所有性能数据
  • GET /performance/api/summary - 获取统计摘要
  • POST /performance/api/record/completion - 记录任务完成

  • GET /alerts/api/statistics - 获取告警统计
  • GET /alerts/api/pending - 获取待处理告警
  • POST /alerts/api/{id}/acknowledge - 确认告警

  • GET /resources/api/today - 获取今日统计
  • GET /resources/api/monthly-stats - 获取月度统计
  • GET /resources/api/agents - 获取Agent排名

认证说明

所有API需要JWT Token认证：

Authorization: Bearer <your-token>

在Swagger UI中点击 "Authorize" 按钮输入Token

响应格式

成功响应：

{
  "status": "success",
  "data": { ... }
}

错误响应：

{
  "status": "error",
  "message": "错误信息"
}
