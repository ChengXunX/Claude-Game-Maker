Agent 能力与技能管理

能力 (Capability) — Agent 能做什么（系统操作）
如：创建Agent、提交代码、发送任务。通过结构化JSON调用，有执行器，敏感能力需审批。
技能 (Skill) — Agent 怎么做更好（知识/经验）
如：代码审查技巧、Git提交规范。注入prompt增强AI能力，可自动学习。
