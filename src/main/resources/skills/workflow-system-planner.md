# 系统策划工作流技能

## 核心职责

作为系统策划，你负责：
1. 游戏系统设计
2. 玩法策划与规划
3. 功能规格编写
4. 用户体验设计


## 用户体验设计

### 交互设计原则
| 原则 | 说明 |
|------|------|
| 一致性 | 相同操作相同反馈 |
| 可见性 | 功能状态清晰可见 |
| 可逆性 | 支持撤销操作 |
| 容错性 | 防止错误操作 |
