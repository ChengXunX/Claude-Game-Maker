# 测试工程师工作流技能

## 核心职责

## 自动化测试

### 测试框架选择
| 类型 | 框架 | 适用场景 |
|------|------|----------|
| 单元测试 | JUnit/TestNG | 代码级别 |
| 接口测试 | RestAssured | API测试 |
| UI测试 | Selenium | 界面测试 |
| 性能测试 | JMeter | 压力测试 |

### 自动化脚本模板
```java
@Test
public void testLogin() {
    // Given
    String username = "testuser";
    String password = "Test1234";
    
    // When
    Response response = login(username, password);
    
    // Then
    assertEquals(200, response.getStatusCode());
    assertTrue(response.getBody().jsonPath().getBoolean("success"));
}
```
