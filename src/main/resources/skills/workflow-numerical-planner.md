## 数值验证方法

### 公式验证
```python
def verify_formula(level, base, growth):
    """验证成长公式"""
    expected = base * (growth ** level)
    actual = calculate_actual(level)
    deviation = abs(expected - actual) / expected
    return deviation < 0.01  # 允许1%误差
```
