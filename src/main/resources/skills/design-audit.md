# 前端设计审查技能

> 对现有页面进行设计质量审计，识别"AI 味"和通用模板问题
> 输出结构化的审查报告，每条带具体修改建议

## 审查流程

1. **扫描** - 读取页面代码，识别框架（Vue/Element Plus）、样式方案
2. **诊断** - 按维度逐项检查，标记问题
3. **输出** - 给出审查报告，按严重程度排序

## 审查维度

### 1. 排版问题

**检查项：**
- [ ] 使用浏览器默认字体（未声明 font-family）
- [ ] 所有文字都是同一个字号（缺乏层次）
- [ ] 标题和正文字号差距太小（< 1.5 倍）
- [ ] 行高过小（< 1.4）或过大（> 2.0）
- [ ] 等宽数据用了比例字体（数字对不齐）
- [ ] 段落过宽（> 80 字符/行）
- [ ] 中英文混排没有加空格

**修复模板：**
```css
/* 问题：所有文字都是 14px */
/* 修复：建立层次 */
.page-title { font-size: 28px; font-weight: 700; letter-spacing: -0.02em; }
.section-title { font-size: 18px; font-weight: 600; }
.body-text { font-size: 14px; line-height: 1.6; }
.caption { font-size: 12px; color: var(--el-text-color-secondary); }
.data-number { font-family: 'JetBrains Mono', monospace; font-variant-numeric: tabular-nums; }
```

### 2. 色彩问题

**检查项：**
- [ ] 大面积使用 Element Plus 默认蓝色
- [ ] 使用纯黑 `#000000` 作为文字色
- [ ] 渐变背景用了 AI 常用的紫蓝配色
- [ ] 颜色过多（超过 3 种强调色）
- [ ] 深色主题用了纯黑背景 `#000`
- [ ] 浅色主题文字对比度不足
- [ ] 状态色不一致（同一状态在不同地方颜色不同）

**修复模板：**
```css
/* 问题：背景 #000，文字 #fff */
/* 修复：柔和的深色 */
background: #0a0a0f;
color: #e8e8e8;

/* 问题：Element Plus 蓝色铺满 */
/* 修复：减少蓝色面积，只用于关键操作 */
.el-button--primary { /* 仅保留主操作按钮 */ }
/* 其他元素用中性色 */
```

### 3. 布局问题

**检查项：**
- [ ] 所有内容居中对齐
- [ ] 三列等分卡片（AI 最常见的布局）
- [ ] 没有最大宽度约束（宽屏拉伸）
- [ ] 卡片间距过小（< 12px）
- [ ] 卡片强制等高（flex stretch）
- [ ] 没有视觉焦点（所有元素同等重要）
- [ ] 移动端没有适配

**修复模板：**
```css
/* 问题：三列等分 */
/* 修复：不对称布局 */
.dashboard-grid {
  display: grid;
  grid-template-columns: 2fr 1fr;  /* 左宽右窄 */
  gap: 20px;
}

/* 问题：没有最大宽度 */
/* 修复：添加容器约束 */
.content-container {
  max-width: 1200px;
  margin: 0 auto;
  padding: 0 24px;
}
```

### 4. 组件问题

**检查项：**
- [ ] 直接使用 Element Plus 默认样式（未定制）
- [ ] 所有卡片一模一样（没有视觉变化）
- [ ] 表格列数过多（> 8 列）
- [ ] 按钮大小不一致
- [ ] 空状态使用默认的"暂无数据"
- [ ] Loading 状态缺失
- [ ] 错误状态缺失

### 5. 动效问题

**检查项：**
- [ ] 没有任何过渡效果
- [ ] 有无限循环的装饰动画
- [ ] 过渡时间过长（> 500ms）
- [ ] 使用了 `all` 作为 transition 属性（性能差）
- [ ] 动画使用了 `top/left/width/height`（触发重排）

### 6. 间距问题

**检查项：**
- [ ] 间距不一致（8px、10px、12px 混用）
- [ ] 卡片内边距过小（< 16px）
- [ ] 区块之间没有呼吸感（间距 < 24px）
- [ ] 表单项间距不统一
- [ ] 移动端间距未缩小

## 输出格式

```
## 设计审查报告

### 严重问题（必须修复）
1. [色彩] 全局使用 Element Plus 默认蓝色，缺乏品牌感
   → 建议：定义品牌色，只在主操作按钮使用强调色

### 中等问题（建议修复）
2. [布局] 仪表盘使用三列等分卡片，无视觉层次
   → 建议：改为 2:1 不对称布局，主图表占 2/3 宽度

### 轻微问题（可选优化）
3. [间距] 卡片内边距 16px，略显紧凑
   → 建议：增加到 24px，提升呼吸感

### 优点
- 色彩统一，没有杂乱的强调色
- 移动端适配完善
```
