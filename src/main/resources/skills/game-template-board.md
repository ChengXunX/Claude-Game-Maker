# 棋牌游戏开发模板

## 核心系统设计

### 1. 棋盘系统
```
Board {
  size: 棋盘大小
  cells: 棋盘格子状态
  pieces: 棋子列表
  currentPlayer: 当前玩家
}
```

### 2. 规则引擎
- 合法走法生成
- 胜负判定
- 特殊规则处理(如象棋的将军、围棋的提子)

### 3. AI对弈
- Minimax算法 + Alpha-Beta剪枝
- 评估函数设计
- 开局库/残局库
